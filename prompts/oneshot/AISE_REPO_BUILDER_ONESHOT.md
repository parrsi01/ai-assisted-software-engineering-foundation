# One-Shot Prompt: AI-Assisted Software Engineering Repo Builder

Use this when you want an AI agent to generate or upgrade a certification-level AI-assisted coding learning repository.

## Prompt (Template)

`You are a senior curriculum engineer and repository architect. Build a professional AI-Assisted Software Engineering training repository (not "vibe coding") from beginner to advanced. Use a certification-level structure with: root README, docs index, offline index, project manual, lesson execution companion, lesson research companion, detailed runbooks, Library/ reference theory notes, prompts/ templates, projects/ hands-on labs, tickets/ Jira-style drills (scenario/reproduce/debug/root-cause/fix/reset), scripts/ validators, tests/, and .github/workflows for lint + CI. Include modules for LLM foundations, prompt software engineering, context engineering, coding agents, evals, RAG for coding assistants, multi-agent workflows, AI code review, security/governance, and enterprise adoption. Use mobile-friendly Markdown, explicit acceptance criteria, and reproducibility-focused instructions. Add professional LinkedIn role terminology (AI-Assisted Software Engineer / Agentic Software Engineering Practitioner). Ensure local validation and CI pass.`

## Use Parameters

- Target language(s)
- Repo constraints (offline-safe / no network / enterprise policy)
- Tooling stack (Python, Node, etc.)
- Certification level (foundation / practitioner / advanced)
