# Python Projects

Structured Python exercises and solutions.

| Module | Focus |
|--------|-------|
| 01_fundamentals | Variables, types, control flow, functions, I/O |
| 02_data_structures | Lists, dicts, sets, stacks, queues, linked lists, BST |
| 03_algorithms | Sorting, searching, recursion, memoization |
| 04_oop_and_design_patterns | Classes, inheritance, design patterns |
| 05_functional_programming | Higher-order functions, closures, decorators |
| 06_concurrency | Threading, multiprocessing, asyncio |
| 07_systems_and_networking | Sockets, HTTP, subprocess, regex |
| 08_data_science_and_algorithms | NumPy, pandas, statistics, time series |
| 09_advanced_algorithms | DP, graphs, tries, segment trees, union-find |
| 10_capstone_projects | Project ideas and sample project |

## How to use
Each module has `exercises.py` (stubs to complete) and `solutions.py` (reference implementations). Read the docstring for each exercise, implement the function body, then compare with `solutions.py`.

```bash
python3 01_fundamentals/exercises.py   # run exercises (some have built-in tests)
python3 01_fundamentals/solutions.py   # run solutions
```
