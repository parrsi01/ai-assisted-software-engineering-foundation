# Agent Engineering Manual

## Agent Building Blocks

- Model
- System prompt / policy
- Tool interface
- Memory / state
- Planner (optional)
- Executor loop
- Verifier / evaluator
- Human approval checkpoints

## Agent Types

- Single-step assistant
- Tool-calling coding agent
- Planner/executor agent
- Reviewer/critic agent
- Multi-agent orchestration system

## Failure Modes

- wrong plan
- wrong tool selection
- stale context
- unsafe command proposal
- silent assumption drift
- unverified completion
