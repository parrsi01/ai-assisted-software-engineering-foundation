# Team Operating Model Manual

## Roles in AI-Native Delivery Teams

- AI-Assisted Software Engineer (builder)
- Agent Workflow Engineer (automation/orchestration)
- AI Code Reviewer / Quality Engineer
- Platform/DevEx Engineer (tooling and guardrails)
- Security Reviewer / Governance Owner

## Team Standards

- define prompt templates centrally
- version agent policies
- track prompt/eval regressions
- require human approval for high-risk actions
- maintain incident tickets for AI failures
