# AI-Assisted Software Engineering Reference Library

> A structured, course-wide reference of the concepts,
> workflows, risks, and engineering patterns used in this repository.
> Written for beginners. Formatted for mobile reading.

---

## How to Use This Library

Each file covers one topic area from beginner to advanced.
Use the same reading pattern across all files:

- what it is
- why it matters
- how it appears in real AI coding workflows
- common failure modes
- how to verify success
